# 智慧商城（Vue2 移动端商城）

基于 Vue2 + Vant UI 的移动端电商 Demo，支持商品浏览、购物车、订单结算、用户登录等功能。

## 线上地址

👉 [点击预览](https://69c3a72ac4f810221bd5d410--luxury-stroopwafel-7fe4ff.netlify.app/#/home)

## 技术栈

- **Vue 2** + Vuex + Vue Router
- **Vant UI** 组件库
- **Axios** 请求封装
- **Less** 样式预处理器
- **Git** 版本控制

## 功能模块

- 首页轮播图、商品推荐
- 全部分类 + 商品列表
- 商品详情（图片轮播、评价、加入购物车）
- 购物车管理（数量增减、全选、删除）
- 订单结算（地址选择、提交订单）
- 用户登录（手机号 + 验证码）
- 我的订单（不同状态筛选）

## 本地运行

```bash
# 克隆项目
git clone https://github.com/你的用户名/仓库名.git

# 进入项目目录
cd 仓库名

# 安装依赖
